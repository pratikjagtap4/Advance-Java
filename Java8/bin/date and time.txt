there are 2 main date and time api in java 

1> java.util.Date & java.util.calendar - this are legacy Api introduce in previous verions of java. 
                             drawbacks:- it is not thread safe and difficult to handle time zone

> java.time :-  this is a newer api introduce in java 8. it addresses the short comings of the legacy api 
                by providing the following:
                1. immutable classes for date and time representation 
                2. easier time zone handling.
                3. thread safety
